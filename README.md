# Txt2Excel

编辑于 2020.2.14
###############################

本程序用于实现批量Txt文档转Excel格式（限 .xlsx格式）

使用步骤如下：
1、解压压缩包到任意目录下
2、打开dist文件夹，确认是否存在两个文件夹分别叫 input 和 output （不能是别的名称，注意全为小写）
3、将要转换的Txt文件放入 input 文件夹内，双击运行 Txt2Excel.exe
4、转换后的Excel文件在output文件夹下

notes：txt里的内容按照行和列用空格隔开，例如：

1 2 3 4
5 6 7 8
中文 English X
